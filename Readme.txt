Prosjektbeskrivelse: 
Jeg valgte å lage en nettside om meg selv. Nettside består av 7 sider: hovedside, om meg, mine prosjekter, lenker, bøker og kontakt side. Selve nettside er tenkt for å bruke som Portfolio som jeg kan vise til fremtidige arbeidsgivere. De kan lett å finne informasjon om meg, mine prosjekter og kontakte meg på lett og rask måte.
Side om meg inneholder tekst om meg selv og slider med bilder fra livet mitt. Neste side er om mine prosjekter i programmering. Side med lenker gir mulighet for andre medstudenter og nybegynnere i programmering å bli kjent med andre nettsteder som tilbyr nyttige ressurser. Side med bøker viser tabellene med bøker som jeg leser hvert år. Som kan bli godt eksempel hva er lurt å lese for å bli god i programmering.
På kontakt side kan bruker kontakte meg ved å fylle et skjema. Det består av fornavn, etternavn, alder, adresse, mobilnummer og melding. Skjema har validerings kode som viser bruker alert meldinger hvis det er noen plasser ikke fylt, eller fylt feil. Nederste på nettside kan man finne mine Facebook kontakter og min CV.

Web design:
Jeg prøvde å lage min egen nettside med responsiv og brukervennlig design. Det er viktig å designe slik side at det blir pent resultat som er enkelt for brukeren å navigere på. Det er også viktig at en nettside får et design som er klassisk nok i sin struktur til å gjøre den moderne med enkle grep uavhengig av trenden i tiden. Det innebærer at jeg kan bytte bilder og andre elementer ved fornyelse. Jeg har valgt å bruke grå og oransje toner som gir god kontrast med hverandre og det er lettere å lese tekst for brukere. Brukere også kan lettere å se hvor navigasjons valgbar er osv. Innholdet er strukturert på en hensiktsmessig måte som gjør det lett for brukeren å finne det de er på jakt etter.
 Responsiv design betyr at brukeren ikke behøver å bry seg om hvilken enhet som brukes for å lese nettsiden. Bruker kan enkelt bruke mobile løsninger som nettbrett, bærbar pc eller mobil i stedet for stasjonær pc. Ved responsivt design så tilpasser nettstedet seg etter enheten som aksesserer sidene og tilpasser skjermbildet til dette. 


Teknologi brukt for implementasjon: 
Jeg har brukt HTML5, CCS3, Javascript, Bootstrap5 og ganske mye W3.CSS. Jeg har også publisert denne nettsiden på min Github side.
For å skrive koden brukte jeg Visual Studio Code som er en kildekoderedigerer laget av Microsoft med Electron Framework, for Windows, Linux og macOS. Funksjoner inkluderer støtte for feilsøking, syntaksutheving, intelligent kodefullføring, utdrag, koderefaktorering og innebygd Git.
